# Sophia Stack v1.5.0 — "anybody can be a dev" 🚀

**Build websites and apps by chatting with AI — then own everything.**

Sophia Stack is an **open-source, self-hosted, provider-agnostic AI app builder**. You get the
Lovable/Base44-style "describe it and watch it build" experience — but it runs on *your* hosting, with
*your* AI provider (or a local model), and the deployed site is **agent-operable** over REST/MCP/OpenAPI.
No SaaS lock-in. No vendor handcuffs. It's yours.

v1.5.0 is the **"anybody can be a dev"** release. A non-technical person can go from a downloaded zip to
a live site in minutes — and a developer can wire Codex/Claude Code/Cursor straight into it.

## ✨ What's in it

- **🪄 Lovable-style build chat** — describe what you want and watch the AI build it, **streaming
  token-by-token**, with a **live preview (VEX)** beside the chat and a **🚀 Push to Live** button.
- **🧠 Bring your own AI** — OpenAI, Anthropic, Gemini, OpenRouter, Groq, Mistral, **local models**
  (Ollama / LM Studio / vLLM), or any custom endpoint. Swap anytime. Built-in vector memory.
- **🤖 Agent-operable** — the *deployed* site speaks **REST + MCP + OpenAPI**, so any agent
  (Claude Code, Codex, Cursor, Grok…) can read and edit it live — and the edits stream into the preview.
- **🛡️ Safe by construction** — every AI edit is **validated before commit**, snapshotted, and
  **one-click reversible**. An immutable core. Your AI can't break your site.
- **🧩 Community extensions** — a real plugin system with scoped permissions, hooks, **its own admin
  panel**, and **one-click install straight from a public git repo**. Scaffold one in seconds:
  `sophia create-extension my-tool`. *(The Sophia SEO Suite is the first major one.)*
- **🔐 Members + payments** — built-in end-user accounts, and connect **your own** Stripe to sell to
  your customers (Sophia takes no cut).
- **♻️ Safe auto-updates** — one-click **Update now**: backup → swap → health-check → **auto-rollback**
  if anything's wrong. Your data is never touched and migrates forward automatically.
- **📦 Runs anywhere** — a zero-install single artifact. `sophia start`, `docker compose up`, or upload
  the zip to any Node host. No VPS gymnastics.
- **✅ Solid** — 235 automated tests across 22 suites. Apache-2.0. Self-hosted. Yours.

## 🏁 Get started

```bash
git clone https://github.com/Chorozion/Sophia-Stack.git
cd Sophia-Stack && npm install && npm run build
sophia start          # opens your browser → onboarding wizard
# or: docker compose up    # or upload release/sophia-stack.zip to any Node host
```
